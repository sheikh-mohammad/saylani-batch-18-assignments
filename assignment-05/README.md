# **Assignment Title:** User Ki Age Ki Hissab - Days Aur Months Show Karna

## Objective:

User se age input lena aur uski age ko detail me explain karna (years, months, days).

## Task Instructions:

- HTML me ek input field create karo jahan user apni age (years me) enter kare.
- Ek button banao jiska text ho: "Show My Age Detail".
- JavaScript me ek program create karo jo button ke click par call ho.

- Input field se user ki age get karo.
- Agar input me number ke ilawa koi or type ho to alert show ho: "Invalid age".
- Agar valid age ho to age ko months aur days me convert karo.

- 1 year = 12 months
- 1 year = 365 days
- Result ko console par show karo.

### Output Format:

```
You are 17 years old
That is equal to:
204 months
6205 days
```
